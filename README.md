# 🏦 Batumionline.biz — פלטפורמת פתיחת חשבון בנק בגאורגיה

## הרצה מהירה (מקומית)

```bash
# 1. כנס לתיקיית הפרויקט
cd ~/Desktop/Batumionlinebiz_CursorProject

# 2. התקן dependencies
npm install

# 3. ערוך את ה-.env (ראה מטה)
# (קובץ .env כבר קיים — יש לעדכן את הטוקנים)

# 4. הרץ
npm start
```

**כתובות:**
- 🌐 דף נחיתה: http://localhost:3000
- 👤 פאנל לקוח: http://localhost:3000/client
- ⚙️ פאנל מנהל: http://localhost:3000/admin  (סיסמה: admin123)
- ⚖️ פאנל עו"ד: http://localhost:3000/lawyer  (סיסמה: lawyer123)

---

## קובץ .env — מה צריך לעדכן

```env
# סיסמאות — שנה לפני פרסום!
ADMIN_PASSWORD=הסיסמה_שלך
LAWYER_PASSWORD=סיסמת_עוד

# Green API - WhatsApp
GREEN_API_TOKEN=הטוקן_שלך  ← חובה לוואטסאפ

# Google Drive (אופציונלי — יעבוד בלעדיו)
DRIVE_FOLDER_PASSPORT=ID_תיקייה
DRIVE_FOLDER_DOCUMENTS=ID_תיקייה
DRIVE_FOLDER_POA=ID_תיקייה

# Morning - תשלומים (אופציונלי — אפשר לאשר ידנית מהאדמין)
MORNING_API_KEY=המפתח_שלך

# URL בפרסום
APP_URL=https://batumionline.biz
```

## זרימת השלבים

| שלב | מסך | פעולה |
|-----|-----|--------|
| 1 | הרשמה | לקוח ממלא פרטים + מעלה דרכון |
| **2** | **⏳ ממתין לאישור דרכון** | **מנהל מאשר/דוחה** |
| 3 | תשלום | לקוח משלם ₪1,900 |
| 4-5 | נוטריון | הורדת ייפוי כח + הנחיות |
| 6 | צ׳ק ליסט | לקוח מסמן 7 מסמכים מוכנים |
| 7 | העלאת מסמכים | 7 PDFs לגוגל דרייב |
| 8 | משלוח | לקוח מזין מספר מעקב |
| **9** | **⏳ ממתין לאישור קבלה** | **עו"ד מסמן "הגיעו"** |
| 10 | בנק בתהליך | עו"ד מסמן "נפתח" |
| 11 | משוב | לקוח מדרג |

### כניסה חוזרת
- כניסה דרך לינק (`/client?phone=...`) — מגיע לשלב הנוכחי
- כניסה דרך טלפון — מגיע לשלב הנוכחי
- בשני המקרים — אם הוא במסך המתנה יראה את מסך ההמתנה

## פאנל מנהל
- רשימת כל הלקוחות
- אישור / דחיית דרכון (שולח הודעת וואטסאפ + לינק תשלום)
- אישור תשלום ידני (אם Morning לא מוגדר)
- ביטול חסימה

## פאנל עו"ד
- רשימת לקוחות בשלב המשלוח
- סימון "מסמכים הגיעו" → לקוח עובר משלב 10 → 11
- סימון "חשבון נפתח" → לקוח עובר משלב 11 → 12

---

## פריסה ל-Render.com

1. דחוף לגיטהאב (repository חדש)
2. render.com → New Web Service → חבר ל-GitHub repo
3. Build Command: `npm install`
4. Start Command: `npm start`
5. Environment Variables: הוסף את כל .env
6. חבר דומיין: batumionline.biz → Settings → Custom Domain
